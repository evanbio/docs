## 📖 关于Logbook

**Logbook** 是我的个人知识管理系统——精心整理的教程、方法论、工具配置和研究资源集合。

> "电影以余味定高低 ✨"
> "记录是成长的开始"

这个知识库记录了我在学习和工作中积累的技术知识、研究见解和实用指南。它被设计为一个可搜索、有组织的参考资料库，不仅服务于我自己，也希望能帮助到觉得这些资源有用的人。

🔗 **在线访问**: [https://docs.evanzhou.org/](https://docs.evanzhou.org/)

## 📂 内容分类

### 🛠️ 技术编年史
日常开发工作的技术教程和工具配置。

- **Linux & Bash** - 命令行基础和Shell脚本
- **Web技术** - CSS、HTML基础和最佳实践
- **效率工具** - Pandoc、Chrome扩展和工作流优化

### 📈 R语言视界
R编程用于数据分析和可视化。

- **R编程** - 从基础到高级技巧
- **数据可视化** - 使用ggplot2创建引人注目的图表
- **统计分析** - 实用的统计方法和应用

### 🗒️ 思维笔记
精选的名言、见解和灵感片段。

- **语录收藏** - 来自书籍、电影和生活的智慧
- **思考碎片** - 值得保存的想法

### 🧠 创意资源
命名、设计和创意灵感的资源。

- **命名规范** - 项目和变量命名最佳实践
- **设计灵感** - 配色方案、字体排版和UI模式

### 📘 标准操作流程 (SOPs)
常见任务和账户设置的分步指南。

- **账户注册** - 各种平台的详细注册指南
- **配置教程** - 工具和服务的设置说明

### 🩺 医学研究
生物医学数据库、研究工具和科研资源。

- **数据库参考** - PubMed、NCBI和专业数据库
- **研究工具** - 生物医学研究的软件和平台

### 💰 财富蓝图
财务规划、投资策略和移民资源。

- **个人理财** - 预算和投资原则
- **移民指南** - 国际流动性资源

### 🧬 数据图谱
生物信息学、数据分析和计算生物学资源。

- **生物信息学工具** - 基因组分析的软件和流程
- **数据分析** - 生物学数据的方法和最佳实践

## ✨ 站点特性

- 📚 **Gitbook主题** - 清爽的书籍式阅读体验
- 🌙 **深色模式** - 任何光线下都舒适的阅读
- 🔍 **Algolia搜索** - 快速、强大的全文搜索 (Ctrl+K)
- 📊 **多种视图** - 表格、画廊和看板布局
- ⌨️ **键盘快捷键** - 使用热键高效导航
- 🔄 **持续更新** - 内容定期优化和扩充
- 🌐 **多设备支持** - 响应式设计适配任何屏幕

## 🛠️ 技术栈

这个知识库基于 [NotionNext](https://github.com/tangly1024/NotionNext) 构建，使用Notion作为内容管理后端。

**核心技术**:
- **框架**: Next.js + React
- **数据源**: Notion API
- **主题**: Gitbook (定制版)
- **样式**: Tailwind CSS
- **部署**: Vercel
- **搜索**: Algolia
- **分析**: Google Analytics
