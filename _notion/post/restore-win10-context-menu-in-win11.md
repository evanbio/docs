### 问题描述

切换到 Windows 11 后，最大的痛点之一就是**右键菜单**的设计。

微软将原本常用的功能折叠进了二级菜单，绝大多数情况下（例如使用 7-Zip、VS Code 或 Git），我们都需要多点击一次 **“显示更多选项” (Show more options)** 才能找到所需功能。虽然可以使用快捷键 `Shift + F10`，但在高频操作下，这依然远不如原本的直接点击来得高效和符合直觉。

本文记录如何通过修改注册表，将 Win11 的右键菜单恢复为 Win10 的经典样式。

---

### 解决方案

### 1. 打开终端

使用快捷键 `Win + X` 唤起系统快捷菜单，选择 **Windows 终端（管理员）** 或 **Terminal (Admin)**。

### 2. 执行注册表修改命令

在弹出的命令行窗口中（PowerShell 或 CMD 均可），复制并执行以下命令：

```bash
reg add "HKCU\Software\Classes\CLSID\{86ca1aa0-34aa-4e8b-a509-50c905bae2a2}\InprocServer32" /f /ve
```

> 提示：直接在终端窗口点击鼠标右键即可粘贴，按下回车键，显示“操作成功完成”即表示修改成功。
> 

### 3. 生效更改

修改注册表后需要重启资源管理器才能生效。你可以选择重启电脑，或者更快的方案——**重启 Windows 资源管理器**：

1. `Ctrl + Shift + Esc` 打开任务管理器。
2. 在进程列表中找到 “Windows 资源管理器” (Windows Explorer)。
3. 右键选择 **“重新启动”**。

此时再回到桌面或文件夹中点击右键，菜单应该已经恢复为经典的 Win10 样式，不再需要二级折叠。

---

### 如何恢复 Win11 默认样式

如果后续想要体验回 Win11 的新版菜单，只需在终端中执行以下命令删除对应的注册表键值即可：

```bash
reg delete "HKCU\Software\Classes\CLSID\{86ca1aa0-34aa-4e8b-a509-50c905bae2a2}\InprocServer32" /va /f
```

同样需要重启资源管理器生效。
