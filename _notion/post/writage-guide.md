> Writage 是一个 Microsoft Word 插件，安装后可以在 Word 中原生支持 Markdown 文件的编辑与保存。它解决了 Markdown 与 Word 格式间的转换问题，让习惯 Word 的用户也能无缝使用 Markdown。
> 

---

## 1. 什么是 Writage

- **插件性质**：Writage 不是独立程序，而是 Word 的扩展组件。
- **功能定位**：让 Word 直接识别、打开、保存 Markdown 文件（.md），不需要额外的转换命令。
- **兼容性**：保持 Markdown 的语法和样式，确保 Word 与 Markdown 文件之间可互换。

---

## 2. 工作模式

- 安装 Writage 后，Word 中自动启用 Markdown 编辑功能。
- 允许在 Word 中直接保存文件为 `.md` 格式。
- 在 Markdown 与 Word 文件之间切换时，保持语法结构不丢失。
- 无需命令行操作，完全在 Word 图形界面中完成。

---

## 3. 使用场景

- **文档写作**：在 Word 中写文档，同时保证 Markdown 格式兼容，便于跨平台编辑。
- **团队协作**：同一份 Markdown 文件，既能用 Word 打开编辑，也能用代码编辑器维护。
- **科研与出版**：方便在 Markdown / Pandoc / R Markdown 流程中嵌入 Word 工作。

---

## 4. 优势与局限

✅ **优势**：

- 无需命令行，Word 用户零学习成本。
- 格式转换自动完成，支持常用 Markdown 特性。

⚠️ **局限**：

- 功能范围有限，不如专业编辑器灵活（如 Typora、Obsidian）。
- 对复杂 Markdown 特性（如自定义扩展、LaTeX 数学公式）支持较弱。
- 依赖 Word 环境，不适合纯命令行/自动化流程。
