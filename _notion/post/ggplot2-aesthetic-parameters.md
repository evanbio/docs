这些参数通常用于 `geom_*()` 函数中，控制图形的**位置叠加、颜色填充、透明度、边框样式**等，是 `ggplot2` 作图调优的重要组成部分。

---

### 🔧 `position = "identity"`：原位叠加

### ✅ 作用

保持数据的原始位置，不进行 `dodge`、`stack` 或 `jitter` 等位置调整。

适合场景：

- 叠加多个 `geom_*()` 图层
- 自定义透明度、手动控制点/条不偏移

💡 示例：

```r
ggplot(df, aes(x, y)) +
  geom_bar(stat = "identity", position = "identity")

```

**常见组合**：

| position 值 | 效果 |
| --- | --- |
| `"identity"` | 原位不变（默认） |
| `"stack"` | 堆叠显示（条形图默认） |
| `"dodge"` | 水平错开 |
| `"fill"` | 百分比堆叠 |
| `"jitter"` | 防重叠（常用于散点图） |

---

### 🎨 `fill = NA`：无填充，仅边框

### ✅ 作用

将图形内部设置为**透明**（无填充），保留 `color` 指定的边框颜色。

💡 示例：

```r
geom_point(shape = 21, fill = NA, color = "black")

```

- 常用于圈出特定点而不影响其他图层
- 对 `shape = 21–25` 特别有用（可分开设置填充色与边框色）

---

### 🌫️ `alpha = ...`：透明度控制

### ✅ 作用

设置图形元素的透明度（范围 `0 ~ 1`，值越小越透明）

💡 示例：

```r
geom_density(alpha = 1/5, fill = "blue")

```

- 常用于图层叠加时防止遮盖
- 有助于对比多个分布或分类差异
