## 1) 快速上手（Basic Commands）

- **设置启动页面**：`设置 → 启动时 → 打开特定网页或一组网页`，预设常用站点（如 B 站、YouTube）提升开机效率。
- **无痕模式**：`Cmd + Shift + N`（Windows 为 `Ctrl + Shift + N`）开新无痕窗口，避免保存历史/Cookie。
- **同步功能**：`设置 → 您与 Google 同步` 管理同步项；建议在个人设备开启历史、密码、通行密钥等同步，保证多设备一致。

## 2) 常用快捷键（Shortcuts）

- **标签页**：新建 `Cmd + T`｜关闭 `Cmd + W`｜恢复关闭 `Cmd + Shift + T`
- **窗口**：新建 `Cmd + N`｜无痕 `Cmd + Shift + N`
- **页面缩放**：放大/缩小 `Cmd + +/-`｜恢复 100% `Cmd + 0`
- **地址栏**：聚焦 `Cmd + L`
- **清理记录**：`Cmd + Shift + Delete`
- **书签**：收藏当前页 `Cmd + D`｜显示/隐藏书签栏 `Cmd + Shift + B`

> 注：Windows 用户将 Cmd 替换为 Ctrl。
> 

## 3) 进阶技巧（Advanced Tips）

### 3.1 搜索效率倍增

- **自定义搜索引擎**：`设置 → 搜索引擎` 为常用站点设置关键词（如 `bing`），地址栏输入 `bing` + `Tab` 直接搜索。
- **地址栏 @ 快捷搜索**：输入 `@历史记录`、`@书签`、`@标签页` + `Tab` 快速在对应范围内检索。
- **调用 Gemini**：地址栏输入 `@Gemini` + `Tab` 直接对话（可省去打开页面）。

### 3.2 高级搜索语法（Google）

- **精确匹配**：`"关键词"`
- **标题限定**：`intitle:关键词`
- **正文限定**：`intext:关键词`
- **URL 限定**：`inurl:关键词`
- **站点限定**：`site:domain.com`
- **文件类型**：`filetype:pdf/doc/xls/ppt/...`
- **图片尺寸**：`imagesize:宽x高`（如 `imagesize:1920x1080`）
- **时间范围**：搜索结果页 `工具 → 时间` 自定义范围

### 3.3 性能与外观

- **节省内存模式**：开启 `设置 → 性能 → 节省内存`，自动休眠闲置标签页。
- **主题与背景**：右下角“自定义 Chrome”或 `设置 → 外观` 调整主题、上传背景。
- **新标签页快捷方式**：在新标签页“快捷方式”区域添加常用网站（如 Canva），命名后一键直达。

### 3.4 书签与快捷方式

- **快速书签**：右键书签进行分组/重命名；`Cmd + D` 秒收录。
- **书签栏显示**：`Cmd + Shift + B`；建议用文件夹分组（工作/学习/临时）。
- **地址栏快捷方式**：为常访问站点设定自定义关键词（配合 3.1）。

## 4) 常见问题 & 解决方案（Problems & Solutions）

- **内存占用高**：`更多工具 → 任务管理器` 检查各标签页/扩展占用并关闭；必要时开启“节省内存”。
- **无法访问扩展商店**：多见于区域限制，需配合 VPN 访问。
- **受限图片无法保存**：右键 `检查` → 切换到 `Network` → 过滤 `IMG` → 选中对应请求，在 `Headers` 或 `Preview` 找直链保存。
- **下载位置管理**：`设置 → 下载内容` 启用“每次下载前询问保存位置”，避免下载目录混乱。

### 🔑 核心知识点

**① 搜索需求分类**

- 信息资讯（News, Events）
- 知识技能（Tutorials, Concepts）
- 素材文件（Images, Video, Docs）
- 工具软件（Apps, Online tools）

**② 搜索背后的目的**

- Know something → 了解
- Learn something → 学习
- Create something → 创作
- Do something → 执行

**③ 搜索工具对比**

- Google：效率与准确性最佳
- 百度 / 搜狗：局限明显
- 微信“搜一搜”：公众号/视频号生态

**④ 高级搜索语法**

- `"关键词"` → 精确匹配
- `intitle:` → 限定标题
- `intext:` → 限定正文
- `inurl:` → 限定网址
- `site:` → 限定域名
- `filetype:` → 文件类型
- `imagesize:` → 图片尺寸

**⑤ 优质资源渠道**

- **知识/技能**：Google, YouTube, B站
- **素材网站**：Pexels, Pixabay, Mixkit
- **工具网站**：Canva, Remove.bg, VirusTotal, Alternativeto
- **发现相似资源**：SimilarSites

**⑥ AI 与搜索结合**

- ChatGPT、Midjourney 等工具让“搜索 + 学习 + 创作”更紧密
- AI 不仅能回答问题，还能推理、对比、生成内容 → 提升效率
- 趋势：未来搜索将与 AI 融合，演变成“超级搜索技术”
