## 🖥️ 代码编辑器

### RStudio

| 系统 | 字体 | 标注 | 说明 |
| --- | --- | --- | --- |
| Windows | JetBrains Mono | ⭐ | 等宽清晰，支持连字 |
| Windows | Consolas | 🏷️ | 系统自带，经典可靠 |
| macOS | Menlo | 🏷️ | macOS 默认，搭配良好 |

### PyCharm

| 系统 | 字体 | 标注 | 说明 |
| --- | --- | --- | --- |
| Windows | JetBrains Mono | 🏷️ | JetBrains 官方默认推荐 |
| Windows | Consolas | 👍 | 微软经典代码字体 |
| Linux | DejaVu Sans Mono | ⭐ | 等宽适配好，开源 |

### VS Code

| 系统 | 字体 | 标注 | 说明 |
| --- | --- | --- | --- |
| 全平台 | Cascadia Code | 🏷️ | 微软官方，支持连字 |
| 全平台 | Fira Code | 👍 | 开源，连字效果出色 |

> 💡 代码字体建议开启 Ligatures（连字），视觉更流畅
