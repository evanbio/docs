`pmin()` 和 `pmax()` 用于**按元素逐一比较多个向量或列的最小值或最大值**（parallel min/max），是 `min()` / `max()` 的**向量化版本**。

---

### ✅ 基本语法

```r
pmin(..., na.rm = FALSE)
pmax(..., na.rm = FALSE)

```

| 参数 | 含义 |
| --- | --- |
| `...` | 多个向量或列（必须长度一致或可广播） |
| `na.rm` | 是否忽略 `NA` 值（默认 `FALSE`） |

---

### 💡 示例 1：两个向量逐位取最大/最小值

```r
x <- c(1, 5, 3)
y <- c(2, 4, 6)

pmin(x, y)
# [1] 1 4 3

pmax(x, y)
# [1] 2 5 6

```

---

### 💡 示例 2：多个向量比较（支持 3+ 列）

```r
a <- c(1, 3, 5)
b <- c(2, 1, 4)
c <- c(3, 2, 0)

pmin(a, b, c)
# [1] 1 1 0

pmax(a, b, c)
# [1] 3 3 5

```

---

### 💡 示例 3：处理 `NA` 值

```r
x <- c(1, NA, 3)
y <- c(2, 4, NA)

pmin(x, y)
# [1] 1 NA NA

pmin(x, y, na.rm = TRUE)
# [1] 1 4 3

```
