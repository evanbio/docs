## 1. 取并集（union）

```r
union(c(1, 2), c(2, 3))
# 结果: 1 2 3
```

- 返回所有唯一元素（去重合并）

---

## 2. 取交集（intersect）

```r
intersect(c(1, 2), c(2, 3))
# 结果: 2
```

- 返回同时出现在两个向量中的元素

---

## 3. 取差集（setdiff）

```r
setdiff(c(1, 2), c(2, 3))
# 结果: 1
```

- 返回只在第一个向量、但不在第二个向量的元素

---

## 4. 常见应用场景

- 筛选基因集的独特/共有/特异元素
- 判断数据集的唯一性、补集、子集关系
- 批量标签、样本、分组结果对比
- 与 `%in%`、`duplicated()` 等组合用法

---

## 5. 组合操作实例

```r
A <- c(1, 2, 3, 4)
B <- c(3, 4, 5, 6)
only_in_A <- setdiff(A, B)      # 只在A
only_in_B <- setdiff(B, A)      # 只在B
common <- intersect(A, B)       # 两者共有
union_all <- union(A, B)        # 所有唯一元素

```
