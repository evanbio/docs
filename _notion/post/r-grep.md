## 1. grep：返回匹配项的索引

```r
x <- c('Hello', 'World', 1, 2, 3)
grep('o', x)
# 结果: 1 2
# 匹配含有“o”的元素，返回索引
```

- 用于筛选所有匹配元素的**位置**，常见于批量过滤、定位子集

---

## 2. grepl：返回逻辑向量

```r
grepl('o', x)
# 结果: TRUE TRUE FALSE FALSE FALSE
# 判断每个元素是否含有“o”，返回TRUE/FALSE

```

- 常用于**逻辑筛选**，配合下标或子集语法使用

---

## 3. 常见应用场景

- 批量筛选含有某关键词/正则特征的字符串
    
    ```r
    x[grepl('o', x)]   # 返回具体匹配的元素值
    
    ```
    
- 判断哪些元素属于某模式，快速批量过滤

---

## 4. 正则表达式补充

- 支持复杂正则（如“结尾/开头/多字符/数字等”）
    
    ```r
    grep('^H', x)          # 匹配以H开头的元素索引
    grepl('[0-9]', x)      # 是否含有数字
    grep('[0-9]', x, value=TRUE)   # 直接返回匹配元素内容
    
    ```
    

---

## 5. 易错点

- grep返回**位置**（索引），grepl返回**逻辑值**，用法不同
- `value=TRUE`参数可让grep直接返回匹配元素内容
