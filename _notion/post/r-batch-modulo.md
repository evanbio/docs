## 1. 基础算术运算符

- **取余（modulo）：`x %% y`** —— 返回x除以y的余数
- **整除：`x %/% y`** —— 返回x除以y的整数部分（向下取整）

---

## 2. 快速分批分组应用

### 2.1 把1到20分为每组5个

```r
x <- 1:20
batch <- (x - 1) %/% 5 + 1
data.frame(x, batch)

```

- **结果：** 前5个为1组，6–10为2组，以此类推

| x | batch |
| --- | --- |
| 1 | 1 |
| 2 | 1 |
| 3 | 1 |
| 4 | 1 |
| 5 | 1 |
| 6 | 2 |
| ... | ... |
| 20 | 4 |

---

### 2.2 批量给元素分配组标签

```r
labels <- letters[1:4]
group <- labels[((x - 1) %/% 5) + 1]

```

- 1–5为a组，6–10为b组，依此类推

---

### 2.3 判断元素是否为每批的第一个/最后一个

```r
# 每批第一个
x[(x - 1) %% 5 == 0]
# 每批最后一个
x[x %% 5 == 0]
```

## 3. 常见场景

- **批量导入/导出文件，每10个一组自动编号**
- **for循环中分批处理数据/子集**
- **分组统计和分组可视化自动生成分组标签**

---

## 4. 易错与补充

- `%%` 与 `%/%` 是左闭右开的区间（习惯从0开始编号时需减1再分组）
- labels的下标注意越界（总组数需与分组数量匹配）

---

## 5. 实用代码模板速查

```r
# 一批N个分组编号
N <- 7
group <- (1:100 - 1) %/% N + 1

# 自定义分组标签
tag <- c("A", "B", "C", "D")
group_tag <- tag[((1:20 - 1) %/% 5) + 1]

```
